
「聖水ふれんず。」

【概要】
制作ツール：RPGツクールMV
動作環境：OSwin7以降。ツクールMVに準ずる
プレイ時間目安：30分
ED数：3
18歳未満プレイ禁止

【操作方法】
マウス、キーボード、タップでプレイ可能

マウス……
左クリック　決定、メッセージ送り
右クリック　キャンセル、セーブ

キーボード……
Enter、Space、Z等　決定、メッセージ送り
Esc、X等　キャンセル、選択肢が表示されているところで押すとセーブ
各種十字キー　カーソル移動

タップ……　決定、メッセージ送り
選択肢が表示されている項目外をタップするとセーブ可能
